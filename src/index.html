First Start Setup

Hi there! It looks like you're starting the customize tool for the first time. Before you get started, we'd like to ask you to go over the default settings for the tool.

To allow the customize tool to patch your custom item into the preview items, we need the path to your client files. This folder should have the 'Arctium WoW Launcher.exe' in it and it should be the same path as the Freedom Launcher shows under Settings->Install Directory.

This setting determines wether the tool will start WoW after patching the custom item into your client files, so you can immediately preview the item in-game.

This determines what model will be used to display the custom item by default on startup.
